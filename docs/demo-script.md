# Flux — 60-second demo voiceover

Total runtime: 60 seconds. Scene cuts on each `==`. Read at ~155 wpm.

---

**0:00 – 0:06 · Hero shot, dark landing page, cursor blinks in chat box.**

> *"Public energy data is rich, but hard to query. Flux fixes that."*

==

**0:06 – 0:12 · Type the prompt, slowly enough to read along.**

`How did Germany's coal share change between 2015 and 2024?`

> *"Plain English in."*

==

**0:12 – 0:20 · Page transitions to /chat. Mode pill flips to "live". Tool pills appear one by one: `get_generation_mix`, `get_generation_mix`, `make_chart`.**

> *"Claude plans tool calls against an EMBER snapshot — no model-made-up numbers."*

==

**0:20 – 0:30 · A line chart slides in showing Coal halving and Wind doubling. Cursor scrolls to it.**

> *"Real data, real chart. Coal generation halved over the decade. Wind nearly doubled."*

==

**0:30 – 0:38 · Scroll to the Sources panel. Hover the EMBER pill — it lights up.**

> *"Every number has a source pill. EMBER for power data, EU ETS for emissions."*

==

**0:38 – 0:46 · Type a follow-up: "Compare wind generation across France, Spain, and Poland."**

> *"Follow-ups stream the same way. The conversation keeps the same data layer."*

==

**0:46 – 0:54 · A second line chart appears — three coloured lines, France/Spain/Poland.**

> *"Spain led, Poland accelerated, France grew on top of nuclear. All in two seconds."*

==

**0:54 – 1:00 · Cut to outro card: Flux logo, GitHub URL.**

> *"Flux. Talk to the grid."*

## Recording checklist

- [ ] Set the browser to dark mode and 1280×720.
- [ ] Disable autocomplete on the chat input (already disabled in code).
- [ ] Use `ANTHROPIC_API_KEY` in `.env.local` so the mode pill reads `live`.
- [ ] Record the type-ins at 1.5× speed in the editor and slow them in post.
- [ ] Cut at the highlighted points; add a 6 BPM background pad under the VO.
- [ ] Export 1080p MP4 + a 6-second hero GIF for the README.
